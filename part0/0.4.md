# 0.4 New note diagram


```mermaid
sequenceDiagram
    participant browser
    participant server
    participant admin
    participant cache

    browser->>server: POST /exampleapp/new_note
    activate server
    server->>admin: Store new note
    activate admin
    admin-->>server: Note stored successfully
    deactivate admin
    server-->>browser: Redirect to /exampleapp/notes
    deactivate server

    browser->>server: GET /exampleapp/notes
    activate server
    server-->>browser: HTML document
    deactivate server

    server->>cache: Update cached notes
    activate cache
    cache-->>server: Cache updated
    deactivate cache

    browser->>server: GET /exampleapp/main.css
    server-->>browser: CSS file

    browser->>server: GET /exampleapp/main.js
    server-->>browser: JavaScript file

    browser->>server: GET /exampleapp/data.json
    server->>admin: Fetch updated notes
    admin-->>server: Updated notes list
    server-->>browser: Updated notes list
